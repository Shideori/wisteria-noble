# Wisteria Noble - Fantasy Outfit Builder

GitHub Pagesで公開するためのアップロード用ファイルです。

## アップロードするファイル

- index.html
- style.css
- script.js
- outfits.json
- README.md

## GitHub Pagesの設定

1. GitHubで新しいリポジトリを作成
2. 上記5ファイルをリポジトリ直下へアップロード
3. Settings → Pages を開く
4. Sourceで「Deploy from a branch」を選択
5. Branchを「main」、フォルダを「/root」に設定
6. Saveを押す
